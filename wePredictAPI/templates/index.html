Info

Welcome to MENG Group Project to visualize the public data from http://fingertips.phe.org.uk/profile/general-practice/data

Infrastructure

The Data is being handled by a MySQL Database. The Data is being extracted by a custom API written in python using the Flask Lib The Frontend section is a HTML5 Application using the following Libs:
  • Bootstrap
  • Bootstrap-slider
  • Morrisjs
  • D3
